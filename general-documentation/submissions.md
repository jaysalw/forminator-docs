# Submissions

{% hint style="info" %}
Submissions are deleted once the **retention limit** is reached (from submission time)
{% endhint %}

Within the management dashboard, you are able to view all available submissions for each form. You are able to execute bulk actions such as:

* Export (to CSV, JSON and PDF)
* Delete

You can also see Analytics and Stats such as number of submissions, IQR, Standard Deviation, Median, Mean, Mode, Average, etc of options from within the form itself.

When viewing a submission you are able to **trigger actions, and view the AI Usage Score** (if enabled).

See [permissions.md](permissions.md "mention") and ensure your permissions are set up correctly.
