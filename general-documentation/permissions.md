# Permissions

* **Admin** \
  All permissions, can edit and manage the server's configuration.
* **Create Forms**\
  Can create forms. Requires `edit_forms` to be able to then edit them.
* **Edit Forms**\
  Able to edit and manage forms.
* **View Submissions**\
  Able to view submissions.
* **Delete Forms**\
  Able to delete forms.
* **Delete Submissions**\
  Allows users to delete a submission (or in bulk). Requires `view_submissions`
* **Edit Settings**\
  Allows users to edit server configuration.
* **Trigger Actions**\
  This permission does not do anything currently, however will require `view_submissions` as well.<br>

Users within your server that have native discord **administrator permissions** and the **server owner** have the administrator permission automatically, though this is not shown within the permissions configuration.
